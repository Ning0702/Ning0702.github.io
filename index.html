Array

What is an Array 固定长度+存储同类数据

关联知识可以包括：array基础 + 使用Arrays 包装类 + ArrayList 可变长度的array

  1. An array is a collection of items.

  2. The items could be integers, strings, objects.

  1. The items are stored in neighboring (contiguous) memory locations. Because they are stored together, checking through the entire collection of items is straightforward.

Primitive operations:

  1. write items in the array

​ 或者直接用 loop（for） 向 array 中写数据。而不能用增强for，因为需要具体的index确定保存位置。

  1. read items from the array

    如果该位置没有内容，就返回默认值，针对此类型是null。（int-0，double-0.0，boolean-false）

​ 或者用loop （for，或增强for）从 array中读数据。

Basic properties of Arrays

Capacity & Length

  1. Capacity 整个array的长度

    从0 到 array.length - 1是其index范围，防止ArrayIndexOutOfBoundsException

    capacity一旦确定，不能改变。

  2. Length 实际存储了多少。可以用length作为记录大小的变量variable

Implementing basic Array operations

1. Inserting Items into an Array

1. Inserting at the End of an Array

​ 就是在空置的array的index中加入数值

2. Inserting at the Start of an Array

​ 将已有数据分别向右移动一位，再将新数据放入第一位。

​ 向右移动时，需要从右向左一位位移动，不然原有数据在复制前就会被覆盖。

3. Inserting Anywhere in an Array

​ 同理于 在数组开头加数据，等同于将要加位置及之后的数据向右移动

2. Deleting Items from an Array

1. Deleting the last element of the Array

​ 并不是真的删除内容，而是可以将覆盖有内容长度的index缩小一位，保证最后一位内容不被遍历到。并且重新写内容时会被覆盖。

2. Deleting the first element of the Array

​ 将全部（除index 0外）的元素全部向左移动一位，并且把有内容的index总数缩小一位。

3. Deletion at any given index

​ 属于上述第二种情况的特例。控制边界，将后续内容向左移动
