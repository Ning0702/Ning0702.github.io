在Github中创建新仓库

本地创建仓库

  1. 打开 Git Bash，并移动到需要上传的文件夹内

  2. git init 创建本地文件夹

关联本地与远程仓库

  1. git remote add origin 地址 ： 即可将两个仓库关联

传入本地仓库

  1. git add .

    将所有文件添加到仓库。注意本地一定要有新文件才能成功。

  2. git commit -m “comment”

传入远程仓库

  1. git push -u origin main

    用于首次上传

  1. git push

    用于以后上传
