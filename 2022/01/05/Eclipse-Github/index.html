Eclipse-Github

Post the project to Github from Eclipse:

  1. Set a repository in Github
  2. Use SSH, instead of the link (DId not figure out the password bug in eclipse)
  3. In Eclipse’s Git Repositories windows, – press “Clone a Git repository and add the clone to the view” – input the SSH
  4. Set a local project in Eclipse
  5. Right click the local project, – “Team” – “Share”
  6. Write code in the local project – right click the project name – “Team” – “Commit”
  7. If change any thing in the Github, should “Team” – “pull..” first, and then “Commit”.
